# Multi-Agent-Research-Team

**Multi-Agent-Research-Team** — это полностью автономная мультиагентная RAG-система, построенная на российской модели **Vikhr-7B-instruct** (GGUF).  

Система имитирует работу настоящей исследовательской команды:
1. **Исследователь** собирает факты из ваших документов.
2. **Аналитик** выделяет ключевые insights.
3. **Критик** проверяет ошибки и предлагает улучшения.
4. **Писатель** создаёт красивый, структурированный отчёт объёмом **2500–3500+ слов**.

Проект специально оптимизирован для запуска в **Google Colab** (бесплатно, без установки на локальный компьютер). Всё работает через удобный Gradio-интерфейс с публичной ссылкой.

**Главное отличие от обычных RAG-чатов**: система не просто отвечает на вопрос — она **пишет полноценный аналитический отчёт** с оглавлением, разделами, таблицами, примерами, рекомендациями и источниками.

## ✨ Ключевые возможности

- ✅ Загрузка неограниченного количества PDF и TXT-файлов
- ✅ Семантический поиск через Qdrant + multilingual-e5-large эмбеддинги
- ✅ 4 независимых агента с защитой от ошибок
- ✅ Генерация **гарантированно длинных** отчётов (2500–3500+ слов)
- ✅ Автоматическое оглавление, Markdown-форматирование, таблицы и списки
- ✅ Кнопка скачивания готового `.md` файла
- ✅ Прогресс-бары в реальном времени
- ✅ Полная совместимость с Google Colab (T4 GPU)
- ✅ Русскоязычные промпты и интерфейс
- ✅ Лёгкая кастомизация (можно изменить промпты под любую тематику)

## 🔄 Как это работает

1. Вы загружаете документы → они автоматически разбиваются на чанки и индексируются в векторной базе **Qdrant**.
2. Вводите тему исследования.
3. Система запускает 4 агента **по очереди**:

   Исследователь → Аналитик → Критик → Писатель
   
5. Писатель генерирует отчёт **по 5 отдельным разделам** (каждый раздел — отдельный вызов модели), благодаря чему достигается большой объём.
6. Готовый отчёт сохраняется и появляется кнопка скачивания.

## 👥 Подробное описание агентов

1. **Researcher Agent** ищет 6–8 фактов из ваших документов. Дополнительно ищет по общим ключевым словам. Возвращает краткий обзор (300–500 слов).

2. **Analyst Agent** выделяет 5 ключевых insights. Строит структуру будущего отчёта. Работает на основе результатов исследователя.

3. **Critic Agent** находит логические ошибки и слабые места. Предлагает конкретные улучшения. Работает как «строгий рецензент».

4. **Writer Agent (главный)** генерирует **5 отдельных разделов**: 
	1. Введение (600–800 слов); 
	2. Анализ исследования (700–900 слов);
	3. Практические рекомендации (600–800 слов); 
	4. Критика и улучшения (500–700 слов); 
	5. Выводы и перспективы (500–700 слов) - Автоматически добавляет оглавление, форматирование и источники.

## ⚠️ Ограничения

- Работает только в Google Colab (из-за размера модели);
- Максимум контекста — 4096 токенов;
- Качество зависит от загруженных документов;
- Модель русскоязычная (английские документы работают хуже);
- Требуется стабильное интернет-соединение для первого запуска.
